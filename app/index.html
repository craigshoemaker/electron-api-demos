Inter-process Communication

The code for ./index.power-monitor.js and ./app/index.js use the webContents.send and ipc APIs to facilitate inter-process communication between the main and renderer process.

Power Monitor

If you have a laptop with a working battery, you can view the power monitor by unplugging and plugging the machine back in again. The developer console will then report the power status.

Use system-wide shortcut command + F12 or control + F12 to return to this window
